How to use these templates

  1. Click a Download button to get the file (replace # with real file links when ready).
  2. Open the file in the appropriate program (Word, PowerPoint, LaTeX editor).
  3. Customize the content: title, authors, section headings, and citations.

Tip: If you plan to host the download files on GitHub, use the raw file URL (or use Netlify file hosting) and paste that URL into the href of the Download button.
